# GuiCreator | The Ultimate In-Game Gui Builder

GuiCreator makes designing user interfaces effortless. No tedious coding is required to build a gui, customize, and manage your menus directly inside an intuitive, in-game interface.

## Key Features

* **In-Gui Editor:** Create and edit complex GUIs directly inside a graphical interface.
* **Item Management:** Add, rearrange, and customize items with a few clicks.
* **Dynamic Actions:** Assign commands or functions to specific items seamlessly.
* **Streamlined Workflow:** Built to save developers and administrators hours of manual configuration time.
* **YML Configuration:** Simple and structured GUI creation via YML files.
* **Versatile Layouts:** Create GUIs in all shapes and sizes.
* **Multiple Inputs:** Includes chat input support and more.
* **Permission Support:** Control access to your menus with custom permissions.
* **Multiple GUI Types:** Seamlessly create and manage various interface types, including enchanting tables, beacons, and custom dialogue menus.
* **Automatic Page Generation:** Automatically creates a new page when you add a row within the GUI.
* **Automatic Saving:** Automatically saves your progress as you create and edit the GUI.

## Supported Plugins

* **Vault:** For economy.
* **PlaceholderAPI (PAPI):** Allows for dynamic placeholders within your GUIs.
